Configs
=====

Installation
------------
Requirements:

  - tmux **`>= 3.1`**
  - awk, perl, sed
  - bash
  - vim
  - curl
  - python3 **`>= 3.6`**, curses
  - universal ctags if you want
  - xclip if you want
  - urlview if you want
  - PathPicker if you want

```
$ cd
$ git clone https://github.com/veyzz/.vcfg.git
$ .vcfg/install.sh
```
