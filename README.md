Configs
=====

Installation
------------
Requirements:

  - tmux **`>= 3.1`**
  - neovim **`>= 0.10`**
  - bash **`>= 4.4`**
  - vim **`>= 8.1`** with +clipboard
  - fzf **`>= 0.56.0`**
  - bat, ag (the_silver_searcher), ripgrep, fd
  - curl, awk, perl, sed
  - python3 **`>= 3.6`**, curses
  - universal ctags if you want
  - xclip if you want
  - tree if you want
  - glow if you want

```
$ cd
$ git clone https://github.com/veyzz/.vcfg.git
$ .vcfg/install.sh
```
