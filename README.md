Configs
=====

Installation
------------
Requirements:

  - tmux **`>= 2.4`**
  - awk, perl, sed
  - bash
  - vim
  - ranger if you want

```
$ cd
$ git clone https://github.com/veyzz/.vcfg.git
$ ln -sf .vcfg/.bashrc && ln -sf .vcfg/.vimrc && \
ln -sf .vcfg/.tmux.conf && ln -sf .vcfg/.tmux.conf.local
```
