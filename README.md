Configs

Installation
------------
Requirements:

  - tmux **`>= 3.1`**
  - neovim **`>= 0.9`**
  - bash **`>= 4.4`**
  - awk, perl, sed
  - vim **`>= 8.1`** with +clipboard
  - curl
  - python3 **`>= 3.6`**, curses
  - universal ctags if you want
  - xclip if you want
  - glow if you want

```
$ cd
$ git clone https://github.com/veyzz/.vcfg.git
$ .vcfg/install.sh
```
